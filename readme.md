## Binance Telegram Feed AutoTrader

- A python bot that automatically trades cryptocurrencies on [Binance](https://accounts.binance.cc/en/register?ref=12326979) global exchange using feeds from telegram channels. 


### Instructions

- Fill the information in the *config.yaml* file.

- run 

  ```
  pip install -r requirements.txt
  python channel_autobuy.py
  ```
